Privacidade e dados pessoais

Política de Privacidade

Esta Política de Privacidade explica como os dados enviados pelo formulário do site do Nutricionista Glaubert Couto são coletados, usados e armazenados.

Quem é o controlador dos dados

O controlador dos dados pessoais coletados por este site é Glaubert Couto.

Quais dados são coletados

O formulário pode coletar nome, WhatsApp, e-mail, cidade, bairro, objetivo principal, preferência de atendimento, melhor horário para contato, mensagem adicional e informações básicas fornecidas voluntariamente sobre acompanhamento nutricional, restrição alimentar, condição de saúde e rotina alimentar.

Para que os dados são usados

Os dados são usados para contato, pré-agendamento, organização inicial do atendimento e acompanhamento da solicitação enviada pelo titular.

Dados sensíveis

Algumas informações fornecidas no formulário podem estar relacionadas à saúde, como restrições alimentares, condições de saúde e rotina alimentar. Esses dados são solicitados apenas para organizar melhor a primeira conversa e não substituem consulta, diagnóstico ou orientação nutricional individualizada.

Base legal e consentimento

O envio do formulário depende do consentimento do titular. Ao marcar o checkbox obrigatório, o titular autoriza o armazenamento e tratamento dos dados informados para as finalidades descritas nesta política.

Compartilhamento de dados

Os dados não são vendidos. Eles podem ser acessados pelo controlador para contato e organização da solicitação. Também podem ser processados por serviços técnicos necessários ao funcionamento do site, como o banco de dados Supabase e o redirecionamento de mensagem pelo WhatsApp.

Onde os dados são armazenados

Os dados enviados pelo formulário são armazenados em banco de dados Supabase configurado para receber os leads do site.

Por quanto tempo os dados são mantidos

Os dados são mantidos pelo tempo necessário para contato, pré-agendamento, organização inicial do atendimento e acompanhamento da solicitação, ou até que o titular solicite a exclusão, observadas obrigações legais aplicáveis.

Direitos do titular

O titular pode solicitar acesso aos seus dados, correção de informações incompletas ou desatualizadas, confirmação de tratamento, revogação do consentimento e exclusão dos dados, quando aplicável.

Como solicitar acesso, correção ou exclusão dos dados

Para solicitar acesso, correção ou exclusão dos dados, entre em contato pelo WhatsApp informado no site. E-mail de contato: [inserir e-mail de contato]. Substitua este placeholder pelo e-mail oficial do controlador quando ele estiver definido.

Segurança das informações

São adotadas medidas técnicas e organizacionais compatíveis com o funcionamento do site para reduzir riscos de acesso indevido, perda ou uso inadequado dos dados. Nenhum sistema digital pode prometer segurança absoluta.

Alterações nesta política

Esta política pode ser atualizada para refletir mudanças no site, no formulário, nos serviços utilizados ou em exigências legais. A versão publicada nesta página será a referência vigente.
